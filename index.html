📈 Ejercicio 1: Media y Varianza Medio

▼

📝 Problema

Se tiene la siguiente distribución de frecuencias de las calificaciones de 30 estudiantes en un examen de estadística:

Intervalo Frecuencia \(f_i\)
[20, 40)5
[40, 60)8
[60, 80)12
[80, 100]5

Calcular:

  1. La media aritmética \(\bar{x}\)
  2. La varianza \(s^2\) y desviación estándar \(s\)
  3. El coeficiente de variación
% Datos del problema
Intervalos: [20,40), [40,60), [60,80), [80,100]
Frecuencias: 5, 8, 12, 5
Total: n = 30

% Fórmulas clave
\bar{x} = \frac{\sum x_i \cdot f_i}{n}
s^2 = \frac{\sum (x_i - \bar{x})^2 \cdot f_i}{n}
CV = \frac{s}{\bar{x}} \times 100
✓ SOLUCIÓN

Paso 1: Marca de clase y cálculos auxiliares

Intervalo \(x_i\) \(f_i\) \(x_i \cdot f_i\) \((x_i-\bar{x})^2 \cdot f_i\)
[20,40)3051505120
[40,60)5084001280
[60,80)7012840480
[80,100]9054503200
Total-30184010080

Paso 2: Cálculo de la Media

\[\bar{x} = \frac{\sum x_i \cdot f_i}{n} = \frac{1840}{30} = \boxed{61.33}\]

Paso 3: Cálculo de la Varianza

\[s^2 = \frac{\sum (x_i - \bar{x})^2 \cdot f_i}{n} = \frac{10080}{30} = \boxed{336}\]

Desviación estándar: \(s = \sqrt{336} = \boxed{18.33}\)

Paso 4: Coeficiente de Variación

\[CV = \frac{s}{\bar{x}} \times 100 = \frac{18.33}{61.33} \times 100 = \boxed{29.89\%}\]

📊 Interpretación: El CV < 30% indica que los datos tienen dispersión media. La media es representativa de la distribución.

\textbf{Media:}
\[\bar{x} = \frac{1840}{30} = 61.33\]

\textbf{Varianza:}
\[s^2 = \frac{10080}{30} = 336\]

\textbf{Desviación:}
\[s = \sqrt{336} \approx 18.33\]

\textbf{CV:}
\[CV = \frac{18.33}{61.33} \times 100 \approx 29.89\%\]

📝 Problema

Un investigador estudia la relación entre el número de horas de estudio semanales (\(X\)) y la calificación obtenida en el examen (\(Y\)). Los datos de 8 estudiantes son:

Estudiante Horas \(x_i\) Nota \(y_i\)
1245
2350

81095

Se pide:

  1. Calcular el coeficiente de correlación de Pearson \(r\)
  2. Determinar la recta de regresión \(Y = a + bX\)
  3. Estimar la nota para un estudiante que estudia 9 horas
  4. Calcular el coeficiente de determinación \(R^2\) e interpretar
% Datos
x = [2, 3, 4, 5, 6, 7, 8, 10]
y = [45, 50, 60, 65, 70, 75, 85, 95]
n = 8

% Fórmulas necesarias
r = \frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum(x_i-\bar{x})^2 \sum(y_i-\bar{y})^2}}
b = \frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}
a = \bar{y} - b\bar{x}
✓ SOLUCIÓN

Medias: \(\bar{x} = 5.625\), \(\bar{y} = 70.625\)

Paso 2: Coeficiente de Correlación

\[r = \frac{314.375}{\sqrt{49.875 \times 2046.875}} = \frac{314.375}{319.47} = \boxed{0.984}\]

✓ Interpretación: Correlación positiva muy fuerte (cercana a 1). A más horas de estudio, mayor calificación.

Paso 3: Recta de Regresión

\[b = \frac{314.375}{49.875} = 6.303\] \[a = 70.625 - 6.303 \times 5.625 = 35.17\]

Recta: \(\hat{Y} = 35.17 + 6.303X\)

Paso 4: Estimación para 9 horas

\[\hat{Y} = 35.17 + 6.303(9) = 35.17 + 56.73 = \boxed{91.9}\]

📌 Predicción: Un estudiante que estudia 9 horas se espera que obtenga aproximadamente 92 puntos.

Paso 5: Coeficiente de Determinación

\[R^2 = r^2 = (0.984)^2 = \boxed{0.968 \text{ o } 96.8\%}\]

📊 Interpretación: El 96.8% de la variabilidad en las calificaciones se explica por las horas de estudio. El modelo es muy bueno para predecir.

\textbf{Coeficiente de correlación:}
\[r = \frac{314.375}{\sqrt{49.875 \times 2046.875}} \approx 0.984\]

\textbf{Recta de regresión:}
\[\hat{Y} = 35.17 + 6.303X\]

\textbf{Estimación para X=9:}
\[\hat{Y} = 35.17 + 6.303(9) \approx 91.9\]

\textbf{Coeficiente de determinación:}
\[R^2 = 0.984^2 \approx 0.968 \text{ (96.8\%)}\]
